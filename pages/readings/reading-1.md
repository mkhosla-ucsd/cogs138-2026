# Reading 1

-  Analysis methods for large-scale neuronal recordings  <a href="https://pubmed.ncbi.nlm.nih.gov/39509504/" target="_blank" rel="noopener">download &#x2197;</a>
 - This paper is a broad introduction to modern methods for analyzing large-scale neural recordings. It covers the main tools researchers use to study population activity, highlights common pitfalls, and gives a useful overview of how neural data science moves from raw recordings to interpretable scientific insights.
